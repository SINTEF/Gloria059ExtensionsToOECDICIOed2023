To run:

change the value of ‘dirCurrent’ to a string with the directory where “EasiSystem Matrix Aggregation” is located.
change the value of ‘dirOriginals’ to a string with the directory  Gloria_ReadMe.xlsx” is located.
change the value of ‘dirReference’ to a string with the directory where the .csv file with the original data, and where the mapping .xlsx files are located

The aggregated .csv file will be created in the ‘dirCurrent’ directory.


OBS. The code is not optimised or clean, so some redundant lines or symbols are to be expected.
